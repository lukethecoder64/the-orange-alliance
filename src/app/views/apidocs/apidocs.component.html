API Docs

The goal of TOA API is to provide us, and potential outside users a way to query our databases.

The Base URL for accessing anything related to our API is http://theorangealliance.org/apiv2

Gaining Access to TOA API

In order to gain access to TOA API, you need to first gain access to an API key. You can apply for an API Key Here. Once you have that key, you must set two custom headers with every GET request. You must set X-Application-Origin to the name of your application. Secondly, you must set X-TOA-Key to your API key.

API Access Levels

Level 0

  • Revoked/not yet approved API key.
  • Every request to TOA API with this key will be denied with a 403 status.

Level 1

  • Single-purpose/team API key
  • Allowed to request data from TOA API, most routes will be available.
  • Every request is tracked and a rate limit of 30 requests/minute is placed.

Level 2

  • Full developer access/application API key
  • Allowed to request data from TOA API
  • Every request is tracked, but there is no limit on rates.
